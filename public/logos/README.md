# Logos Directory

Place collaborator logos in this directory.

## Al-Raazi Hospital Rawalpindi Logo

1. Place the Al-Raazi Hospital logo file here
2. Name it: `al-raazi-logo.png` (or update the path in `src/components/Collaborators.js`)
3. Supported formats: PNG, JPG, SVG
4. Recommended size: 400x300px or similar aspect ratio

The logo will automatically display in the Collaborators section. If the image is not found, a placeholder will be shown.
