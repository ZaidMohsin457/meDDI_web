## ğŸ¯ About meDDI

meDDI is an advanced AI-powered medication safety system that:

- Uses Vision-Language Models (VLMs) to recognize handwritten prescriptions with 90%+ accuracy
- Detects drug-drug interactions (DDIs) in real-time with 95%+ sensitivity
- Provides a mobile application for pharmacists and healthcare providers
- Creates structured prescription databases for pharmacovigilance
- Contributes to SDGs 3, 9, and 17
