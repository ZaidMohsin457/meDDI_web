# Team Photos Directory

Place team member photos in this directory.

## File Naming Convention

- Supervisors: `supervisor1.jpg`, `supervisor2.jpg`, etc.
- Collaborators: `collaborator1.jpg`, `collaborator2.jpg`, etc.
- Developers: `developer1.jpg`, `developer2.jpg`, etc.

## Image Requirements

- Format: JPG or PNG
- Recommended size: 400x400px (square)
- The images will be automatically cropped to circular format
- If an image is not found, initials will be displayed instead

Update the image paths in `src/components/Team.js` to match your file names.
